**Tipo:** AutoU (pré-venda/demonstração para o Itaú)
**Papel:** Desenvolvedor Frontend — em dupla (equipe de 2)
**Status:** Demo entregue
**Stack:** React 19, TypeScript, Vite, Tailwind CSS 4, React Router 7, Leaflet / React-Leaflet (mapas interativos), Vercel, materiais de design do Figma

- Deploy imediato na Vercel para compartilhamento com stakeholders do banco
